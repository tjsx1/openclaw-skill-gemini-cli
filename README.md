# OpenClaw Skill: Gemini CLI

An [OpenClaw](https://openclaw.ai) skill designed for the **Gemini CLI**, enabling autonomous project management and terminal-based coding.

## Features

- **Structural Initialization**: Uses `/init` to generate a `GEMINI.md` project map.
- **Context Injection**: Leverages `@<path>` syntax for precise file-level context.
- **YOLO Workflow**: Supports `Ctrl+Y` autonomy for high-speed execution.
- **Efficiency**: Active token management via `/compress` integration.

## Commands at a Glance

- **Bash Mode**: `!ls -la`
- **Add Context**: `@src/`
- **Stats**: `/stats session`
- **Compression**: `/compress`

## Installation

```bash
openclaw skill install gemini-cli.skill
```
